# 个人简历网站设计文档

**日期：** 2026-04-07
**项目：** 个人介绍静态网站
**状态：** 已批准

---

## 1. 项目概述

### 目标
创建一个简约专业的个人简历静态网站，通过 GitHub Pages 部署。

### 需求确认
- **内容类型：** 个人简历/职业经历
- **视觉风格：** 简约专业风格（干净、现代、商务感）
- **页面结构：** 单页滚动式
- **主要板块：** 个人介绍、工作经历、教育背景、技能专长、项目经验、联系方式

---

## 2. 技术方案

### 技术栈
- **HTML5：** 语义化标签
- **CSS3：** Flexbox 布局、CSS 变量
- **JavaScript：** 平滑滚动导航（少量）

### 文件结构
```
my-website/
├── index.html          # 主页面
├── css/
│   └── style.css      # 样式文件
├── js/
│   └── main.js        # 交互脚本
└── README.md
```

### 选择理由
1. 简单直接，无需构建工具
2. 加载速度快，SEO 友好
3. 易于维护和修改
4. 完美适配 GitHub Pages

---

## 3. 页面布局

### 单页滚动结构

#### 3.1 Hero Section（顶部区域）
- 大标题显示姓名
- 一句话介绍（职位/角色）
- 简洁的导航菜单（固定在顶部）

#### 3.2 About Section（关于我）
- 个人简介
- 照片（可选）
- 核心价值观或职业目标

#### 3.3 Experience Section（工作经历）
- 时间线式展示
- 公司名称、职位、时间段
- 主要职责和成就

#### 3.4 Education Section（教育背景）
- 学校、专业、时间
- 学位信息

#### 3.5 Skills Section（技能专长）
- 技能标签云或分类列表
- 可视化展示（标签形式）

#### 3.6 Projects Section（项目经验）
- 项目卡片展示
- 项目名称、描述、技术栈

#### 3.7 Contact Section（联系方式）
- 邮箱、GitHub、LinkedIn 等
- 简洁的图标链接

---

## 4. 视觉设计

### 色彩方案
| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色 | 深蓝色 | #2c3e50 |
| 背景 | 浅灰色 | #ecf0f1 |
| 强调色 | 蓝色 | #3498db |
| 主文字 | 深灰 | #333 |
| 辅助文字 | 中灰 | #666 |
| 白色 | 白色 | #fff |

### 排版规范
- **主字体：** 系统无衬线字体
  - -apple-system
  - BlinkMacSystemFont
  - "Segoe UI"
  - Roboto
  - sans-serif
- **标题：** 粗体，大字号
- **正文：** 16-18px
- **行高：** 1.6-1.8

### 交互效果
- 平滑滚动到各个区块
- 导航栏滚动时固定在顶部
- 鼠标悬停时的微妙过渡效果
- 响应式设计（移动端友好）

---

## 5. 部署方案

### GitHub Pages 配置
- 使用 `main` 分支的根目录
- 直接访问 `index.html`
- 自定义域名支持（可选）

### 部署步骤
1. 在 GitHub 仓库设置中启用 GitHub Pages
2. 选择 `main` 分支作为源
3. 网站将发布到 `https://stars-chan.github.io/my-website/`
